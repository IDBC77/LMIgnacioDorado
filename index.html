Lenguajes de Marcas

En esta asignatura aprenderás sobre los diferentes lenguajes de marcas, su sintaxis y cómo usarlos para crear contenido estructurado en la web. Estudiarás HTML, XML, y otros lenguajes utilizados para definir la estructura y el formato de los documentos.

Temas de la Asignatura

Tema 1: Introducción a HTML

Aprenderás la estructura básica de HTML y cómo crear páginas web.

Recurso 1: W3Schools - HTML Tutorial
Recurso 2: MDN - HTML

Tema 2: Introducción a CSS

Aquí aprenderás a darle estilo a las páginas web usando CSS.

Recurso 1: W3Schools - CSS Tutorial
Recurso 2: MDN - CSS

Tema 3: JavaScript Básico

Comenzaremos con los fundamentos de JavaScript para agregar interactividad.

Recurso 1: W3Schools - JavaScript Tutorial
Recurso 2: MDN - JavaScript

Tema 4: Introducción a XML

Aprenderás sobre la estructura y uso de XML para representar datos.

Recurso 1: W3Schools - XML Tutorial
Recurso 2: MDN - XML

Tema 5: XSLT y XPath

Conocerás XSLT y XPath para transformar y consultar documentos XML.

Recurso 1: W3Schools - XSLT Tutorial
Recurso 2: W3Schools - XPath Tutorial

Tema 6: JSON y AJAX

Explorarás el formato JSON y aprenderás a trabajar con AJAX.

Recurso 1: W3Schools - JSON Tutorial
Recurso 2: W3Schools - AJAX Tutorial

Tema 7: Avanzando con HTML5 y CSS3

Un enfoque más avanzado de HTML5 y CSS3 para mejorar las páginas web.

Recurso 1: W3Schools - HTML5 Tutorial
Recurso 2: W3Schools - CSS3 Tutorial
